# sca logs

快速初始化

## 用法

sca init

 参数 | 简写 | 必填 | 描述 | 示例
 ------------|------------| ------------|------------|------------
 name|-n|是|生成的项目名称。如果不填写，默认值为 testproject。|-n testproject
 runtime|-r|否|生成的项目运行环境，可选值为python2.7、python3.6、3.7。默认值为 python3.6。	|-r python2.7
output-dir|-o|否|指定项目生成的目录，默认为当前目录。|-o /root/sca/code
